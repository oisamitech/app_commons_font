### Setup excluding styles
Example to exclude solid, regular, duotone

Mac:
```
sh util/configurator.sh --dynamic --exclude solid,regular,duotone
```
Widows:
```
util/configurator.bat --dynamic --exclude solid,regular,duotone
```
